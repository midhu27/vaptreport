# Web Application VAPT Project

## Overview

This project demonstrates a Vulnerability Assessment and Penetration Testing (VAPT) engagement performed as part of cybersecurity training and practical learning.

## Objectives

* Identify security vulnerabilities in web applications.
* Assess the impact of discovered vulnerabilities.
* Provide remediation recommendations.

## Methodology

* Information Gathering
* Enumeration
* Vulnerability Assessment
* Manual Verification
* Risk Analysis
* Reporting

## Tools Used

* Kali Linux
* Burp Suite
* Nmap
* Nikto
* Wappalyzer

## Key Findings

* Cross-Site Scripting (XSS)
* Security Misconfigurations
* Missing Security Headers
* Information Disclosure Issues

## Deliverables

* Professional VAPT Report (PDF)
* Vulnerability Analysis
* Risk Assessment
* Remediation Recommendations

## Disclaimer

This repository is intended for educational and portfolio purposes. Sensitive information, target details, and confidential data have been removed or anonymized.
